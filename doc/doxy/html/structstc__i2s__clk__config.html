Detailed Description

I2S clock configuration.

Note:
I2S ch.0 PLL clock output = (Fin * N * M / K) I2S ch.1 PLL clock output = (Fin * N * M1 / K) I2S ch.0 PLL clock = I2S ch.0 PLL clock output / M I2S ch.1 PLL clock = I2S ch.1 PLL clock output / M1 I2S PLL clock output must be in a certain range, check it in the product datasheet.
