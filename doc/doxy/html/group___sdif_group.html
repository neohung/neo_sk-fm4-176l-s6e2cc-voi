Sdif_Init() initializes a SD host interface instance. Sdif_DeInit() de-initializes a SD host interface instance.

Sdif_EnableInternalClock() Enables SD host internal clock, which makes SD host start work. Sdif_DisableInternalClock() disables SD host internal clock, which makes SD host stops work.

Sdif_EnableSdclk() enables SDCLK output and Sdif_DisableSdclk() disables SDCLK output.

The SD clock is first intialized in Sdif_Init() and can be changed by Sdif_SetClkDiv()

The bit width is first initialized in Sdif_Init() and can be changed by Sdif_SetBusWidth()

The bus speed mode is first initialized in Sdif_Init() and can be changed by Sdif_SetBusSpeedMode()

Sdif_SelBusVoltage() selects SD bus voltage, before Sdif_BusPowerOff() should be called to power off SD bus.

Sdif_BusPowerOn() power on SD bus and Sdif_BusPowerOff() power off SD bus.

Sdif_SendCommand() send the SD command specified by the structure type stc_sdif_cmd_config_t.

Sdif_GetResponse() gets the response data, the count of which is specified by the parameter Sdif_GetResponse::u8ResponseRegCount

Sdif_InitDataTransfer() initializes data transfer, this function must be called before data transfer.

Sdif_ReadData() reads the 32-bit data from data buffer port and Sdif_WriteData() writes 32-bit data to data buffer port.

Sdif_StopAtBlockGap() stops transfer during block gap in multi-block transfer() and Sdif_RestartTransfer() restarts data transfer

Sdif_GetErrStatus() gets the error interrupt status and Sdif_ClrErrStatus() clears error interrupt status.

Sdif_GetNormalStatus() gets the error interrupt status and Sdif_ClrNormalStatus() clears error interrupt status.

Sdif_GetAutoCommandErrStatus() gets the error status of Auto Command.

Sdif_CheckCommandInhit() check whether command without data can be sent. Sdif_CheckCommandWithDataInhit() check whether command with data can be sent.

Sdif_PollCardInsert() polls the card insertion status.

Sdif_SoftwareReset() issues software reset to SD card.
