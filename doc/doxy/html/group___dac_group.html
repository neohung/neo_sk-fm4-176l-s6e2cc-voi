Detailed Description

Provided functions of DAC module:

  • Dac_Init()
  • Dac_DeInit()
  • Dac_SetValue0()
  • Dac_SetValue1()
  • Dac_Enable0()
  • Dac_Enable1()
  • Dac_Disable0()
  • Dac_Disable1()

With Dac_Init() the DAC is configured (10/12-Bit mode, 10-Bit data alignment). Each channel can be en- and disabled by e.g. Dac_Enable0(), Dac_Disable0(). Output values can be set easily by Dac_SetValue0() and Dac_SetValue1().
