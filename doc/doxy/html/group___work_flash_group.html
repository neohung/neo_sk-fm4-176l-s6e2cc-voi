Detailed Description

Provided functions of Work Flash operation:

  • WFlash_ChipErase()
  • WFlash_SectorErase()
  • WFlash_WriteData32Bit()
  • WFlash_WriteData16Bit()

What is Work Flash?

Work Flash have independent area, which is sperated from Main Flash, thus the Flash operation API can be called derectly from Main Flash.

How to use Work Flash module?

WFlash_ChipErase() can erase whole chip space of Work Flash. WFlash_SectorErase() can erase one selected sector. WFlash_WriteData32Bit() writes data into Flash area with word align. WFlash_WriteData16Bit() writes data into Flash area with half-word align.
