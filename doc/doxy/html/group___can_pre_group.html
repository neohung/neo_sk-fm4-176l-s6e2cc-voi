Detailed Description

Provided functions of CAN prescaler module:

  • Canpre_Init() (Initialization of CAN prescaler)

How to use this CAN prescaler module:

This module is activated only when the CAN module is not active and the CAN FD module is active. If the CAN module was activated, use Can_Init() API to initialize the CAN prescaler instead.
