Detailed Description

This section describes the interface for the I2S Unit (I2S)

Provided functions of I2S module:

  • I2s_InitClk()
  • I2s_DeInitClk()
  • I2s_EnablePll()
  • I2s_DisablePll()
  • I2s_CheckPllReady()
  • I2s_StartClk()
  • I2s_StopClk()
  • I2s_Init()
  • I2s_DeInit()
  • I2s_Start()
  • I2s_Stop()
  • I2s_GetStatus()
  • I2s_ClrStatus()
  • I2s_WriteTxFifo()
  • I2s_ReadRxFifo()
  • I2s_GetTxFifoNumber()
  • I2s_GetRxFifoNumber()

I2s_InitClk() initializes I2S clock setting. I2s_DeInitClk() resets the I2S clock setting.

I2s_EnablePll() enables I2S PLL, I2s_DisablePll() disables I2S PLL. I2s_CheckPllReady() checks if the PLL is stable.

I2s_StartClk() starts clock output to I2S macro, I2s_StopClk() stops clock output to I2S macro.

I2s_Init() initializes an I2S instance according its configuration. I2s_DeInit() de-initializes an I2S instance and resets the hardware to initial values.

I2s_Start() starts I2S operation and I2s_Stop() stops the operation.

I2s_GetStatus() gets the I2s status according to the status type. I2s_ClrStatus() clears the I2s status according to the status type.

I2s_WriteTxFifo() writes the I2S TX FIFO by word and I2s_ReadRxFifo() reads the I2S RX FIFO by word.

I2s_GetTxFifoNumber() gets the number of TX FIFO data count and I2s_GetRxFifoNumber() gets the number of RX FIFO data count.

This function initialises one specific I2S module with the parameters provided in the given configuration structure. After successful initialization the I2S module is ready to use.

I2S_Init() has to be called with the parameter pstcConfig of type stc_i2s_config_t, the basic I2S settings automatic retransmission, the I2s baudrate, and the error and status callback function can be set.

All values in pstcConfig have to be in valid range (see i2s.h for allowed ranges of dedicated parameters). The error and status change callback functions can be NULL. In this case no information of error or status changes will be reported to the user application.

To reset and disable the I2S module the function I2s_DeInit() has to be used.
