Usage of example code:

Each example code consists of a 'main.c' and a 'pdl_user.h' module. The main module contains the example code, where 'pdl_user.h' contains all necessary settings and definitions for the example itself. To use and compile an example, just copy these two modules to the 'template/source' folder, overwriting the modules located there. The overwritten modules have backups in the 'template/ source/backup' folder for recovery. The default device selected is MB9BF506R, if other device is used, refer to "fm_pdl_quick_start_guide.pdf" about how to adjust the setting in the IAR and Keil for this device. Follow the explanations and process in the main.c module's description in the comment lines at the top.
